Java project to-do-list
A simple **To-Do List** application written in **Java**, running entirely in the **Command Line Interface (CLI)**.  
It allows users to manage daily tasks by adding, viewing, editing, and deleting tasks easily.
---
# Features
- Add new tasks.  
- View all existing tasks.  
- Edit or delete any task.  
- Simple text-based user interface.  
- Built with Object-Oriented Programming (OOP) principles.
